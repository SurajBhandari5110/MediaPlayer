# MediaPlayer
It is just a simple media player which can play songs and it have all important buttons like play, pause, previous, next.

**
**Green Button-->Play**
**Red Button-->Stop**
**Yellow Button-->Pause**
**Blue Button-->Previous(on left) and Next(on right)**
